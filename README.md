Play Quoridor online at [barricade.gg](https://barricade.gg) or use this package to build your own Quoridor game!

## Description

Quoridor is a strategic board game where players navigate a grid to reach the opposite side while placing walls to hinder opponents. This package provides core game logic, move validation, and state management for building your own Quoridor game.
